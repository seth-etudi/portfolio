# Seth Etudi — Portfolio

This repository contains the personal portfolio website for **Seth Etudi**, showcasing projects, skills, and contact information. The site is built with Vite, React (TypeScript), Tailwind CSS, and Framer Motion.

## Features

- Terminal-style hero section with typing animation
- Projects, Skills, Mentoring, and Contact sections
- Dark mode support with a custom hook
- Interactive UI components using `framer-motion` and `lucide-react`

## Tech Stack

- Vite + React (TypeScript)
- Tailwind CSS
- Framer Motion
- Lucide React (icons)

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

Open http://localhost:5173 (or the address printed by Vite) in your browser.

## Building for Production

```bash
npm run build
```

The production output will be in the `dist` folder.

## Project Structure

- `src/` — application source files
- `src/components/` — React components (Hero, Navigation, Projects, Contact, etc.)
- `src/hooks/` — custom hooks (e.g., `useDarkMode`)

## Customize

Edit content in `src/components/Hero.tsx` and `src/components/Contact.tsx` to update your bio, links, and contact details.
